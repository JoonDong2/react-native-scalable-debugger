# react-native-scalable-debugger

[English](README.md)

플러그인 지향 React Native 디버거 서버를 위한 코어 서버, AppProxy, 플러그인 API 및 `startCommand`입니다.

이 패키지는 `react-native-scalable-debugger` 모노레포의 메인 패키지입니다. React Native를 위한 확장 가능한 디버깅 환경을 구축하는 데 필요한 핵심 구성 요소를 제공하며 다른 플러그인들의 기반 역할을 합니다.

전체 사용법, 엔드포인트 문서 및 플러그인 개발 방법에 대한 자세한 내용은 [모노레포 README](../../README.ko.md)를 참조하세요.
